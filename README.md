# Linux Basic Commands Lab

## 📌 Overview

This lab demonstrates basic Linux commands on an AWS EC2 instance.

## 🎯 Objectives

- Navigate directories
- Create directories
- Create files
- View file contents
- Copy files
- Rename files
- Delete files and directories

---

## Commands Used

| Command | Description |
|---------|-------------|
| `pwd` | Print current directory |
| `ls` | List files |
| `ls -l` | List files with details |
| `ls -la` | List hidden files |
| `mkdir` | Create directory |
| `cd` | Change directory |
| `touch` | Create file |
| `echo` | Write text into file |
| `cat` | Display file contents |
| `cp` | Copy file |
| `mv` | Move or rename file |
| `rm` | Remove file |
| `rmdir` | Remove empty directory |
